# Тест-кейсы для главной страницы onliner.by

## Общие условия
- URL: `https://www.onliner.by/`
- Браузер: Chromium (Playwright)
- Авторизация не требуется

### TC-01. Открытие главной страницы
- **Предусловие:** браузер открыт
- **Шаги:**
  1. Перейти на `https://www.onliner.by/`
- **Ожидаемый результат:** страница успешно открыта, URL содержит `onliner.by`

### TC-02. Проверка заголовка страницы
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Получить `title` страницы
- **Ожидаемый результат:** заголовок содержит `Onliner`

### TC-03. Проверка видимости шапки сайта
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти блок `header`
- **Ожидаемый результат:** шапка отображается

### TC-04. Проверка логотипа и ссылки на главную
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти логотип в `header`
  2. Проверить атрибут `href`
- **Ожидаемый результат:** логотип видим, ссылка ведет на `/`

### TC-05. Проверка верхней навигации
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Получить список ссылок в `header`
- **Ожидаемый результат:** в навигации не менее 5 внутренних ссылок

### TC-06. Проверка поля поиска
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти поле поиска
  2. Проверить, что поле активно для ввода
- **Ожидаемый результат:** поле поиска отображается и доступно

### TC-07. Проверка ссылки на раздел Каталог
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти ссылку на `/catalog`
  2. Проверить доступность ссылки прямым запросом
- **Ожидаемый результат:** ссылка присутствует, ответ сервера успешный

### TC-08. Проверка ссылки на раздел Новости
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти ссылку на `/news`
  2. Проверить доступность ссылки прямым запросом
- **Ожидаемый результат:** ссылка присутствует, ответ сервера успешный

### TC-09. Проверка основного контентного блока
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Найти контейнер `main`
- **Ожидаемый результат:** основной контентный блок отображается

### TC-10. Проверка футера
- **Предусловие:** открыта главная страница
- **Шаги:**
  1. Прокрутить страницу вниз
  2. Найти `footer`
  3. Проверить наличие ссылок в футере
- **Ожидаемый результат:** футер отображается, в нем есть ссылки
